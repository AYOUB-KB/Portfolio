CTFs 🚩

Kerberoasting

Kerberoasting is an Active Directory attack technique that aims to retrieve and crack certain Kerberos tickets offline in order to obtain service account passwords.

Attacking LSASS

Attacking LSASS means targeting the Windows process that stores authentication data. If someone gains access to LSASS, they can extract credentials like password hashes or Kerberos tickets, which can then be reused to move through the network or escalate privileges.
